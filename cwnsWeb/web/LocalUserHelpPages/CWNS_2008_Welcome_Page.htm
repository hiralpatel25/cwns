CWNS 2008 Welcome Page

After logging in, the Local User is directed to the CWNS 2008 Welcome Page (Figure 4-1), which is the gateway to the DEP.  From the Welcome Page, the Local User can:

 

• View assigned facilities and projects in the Facility/Project List

 

• Search for and open assigned facilities and projects

 

• Change his/her password
