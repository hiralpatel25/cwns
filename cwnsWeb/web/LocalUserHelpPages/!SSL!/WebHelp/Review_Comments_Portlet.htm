Review Comments Portlet

The Review Comments Portlet is where communication occurs between the Local User and the State User (Figure 4-8).  The message “Feedback copy does not exist” is displayed if the State has not yet generated the Feedback Copy for this facility/project.  If you have been granted access to provide feedback on the facility/project, contact your State Coordinator and request that they generate a Feedback copy.
