New User Approval

For new user accounts, the Local User Manager is redirected to the User Details portlet to approve or deny user access.  Changes can be made to the Local User’s Access Level(s) and Facility(ies)/Project(s) at this time (see Editing User Details).   By default, new users are given the role Local and View access level for all eligible facilities/projects in their own State.

 

• Click to change the user’s status to Active.  The user receives an email from the account cwns@epa.gov to indicate that have been accepted and to provide access information (e.g., User ID, Password).  

 

To add additional information to the auto-generated e-mail, add text to the box Registration Email Notification Comments before approving the user.

 

• Click to reject the user’s application for access to the DEP.  The user receives an email from the account cwns@epa.gov to indicate that have been denied.  

 

To add additional information to the auto-generated e-mail, add text to the box Registration Email Notification Comments before denying the user.
