EPA Website

The EPA CWNS website (www.epa.gov/cwns) provides resources for Local Users including:

 

• Local User Manual

 

• Small Community Needs Form and resources for completing it

 

• State and EPA CWNS contacts

 

• Other relevant contacts and links

 

• Materials from Web seminar training series (e.g., recorded sessions, presentations, and question and answer documents).  

 

EPA will continue to update this site with information and resources for CWNS throughout the data collection period.
