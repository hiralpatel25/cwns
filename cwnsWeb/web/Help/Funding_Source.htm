Funding Source

The data area Funding Source allows users to view, edit, and update the funding associated with a facility/project.  Existing Funding Sources are list in numerical order by Loan Number.  This optional data area can be used to track funding between Needs Surveys to facilitate adjustments of the needs saved in the Capital Costs data area.

 

Remember: This symbol indicates that a Feedback copy exists for this facility/project.  Click it to view a pop-up box containing the changes made in the Needs by Document data area.  Any changes made to this area will not be reflected on the Feedback version.

 

• Click Add Funding Source to add a new Funding Source to the facility/project.  Funding Sources can also be added using the CWSRF Benefits Reporting Search (CBR) (see next section).  Click  in the Edit column to edit an existing Funding Source (Note: The edit option is not available if the data is from CBR).  Both actions open the Add/Edit Funding Source data area (Figure 9-8).  Explanations of the fields in this data area and in the Funding Source list are in Table 9-9).

 

• Click in the Delete column to remove a Funding Source from the list.

The Funding Source list contains several columns that are relevant only to funding sources that are from CBR, including:

 

• Project : A set of activities that have a single water quality benefit, usually for a single waterbody.  One CWSRF loan may finance several projects.

 

• Cost Category: The CWNS Cost Category for the CWSRF loan.

 

In addition, the column Consistent with CBR indicates whether the information currently entered is consistent with the data in the CWSRF Benefits Reporting (CBR) database.  The column is blank if the data source is not CBR. A is displayed if funding source is consistent with CBR.  If the funding source is inconsistent with CBR, is displayed.
