Facility/Project Types

Facility/Project Types (formerly Natures) define the basic components of a CWNS facility/project (e.g., Treatment Plant, Phase I Municipal Separate Storm Sewer System (MS4), or Agriculture-Croplands). CWNS 2008 Facility/Project Types are provided in Appendix D.  Each type is classified as currently existing (Present), existing in the future (Projected), or both (Present and Projected).

 

• Example: A proposed treatment plant would have a type of Treatment Plant (Projected)

 

• Example: An enlargement to an existing collection system would have a type of Collection: Separate Sewers (Present and Projected)

 

• Example: Abandoning a treatment plant would have a type of Treatment Plant (Present)

The data area Facility/Project Type (Figure 7.3) allows users to view, edit, and update the Facility/Project Type.  At least one Facility/Project Type is required for all facilities/projects.

 

The data area Facility/Project Type (Figure 7.3) allows users to view, edit, and update the Facility/Project Type.  At least one Facility/Project Type is required for all facilities/projects.

 

• Click Add Type to add a new Type to the facility/project.  Click in the Edit column to edit an existing Type.  Either action opens the Add/Edit Type data area (Figure 7-4).  Explanations of the fields in this data area are in Table 7-2.

 

• Click in the Delete column to remove a Type from the list.  Deleting a Type from the facility/project may result in the deletion of data from other data areas, including flow, effluent, discharge, cost, pollution, operation and maintenance, and unit process.

 

A warning message is displayed that indicates what data areas are impacted.  Click Yes to delete the Type and all associated data.  Click No to cancel the delete (Figure 7-3).
