Pollution Problems

The data area Pollution Problems allows users to indicate the pollution problems that the facility/project’s needed capital costs will address.  This data area is enabled for all Facility/Project Types, except Plan Development.  It is an optional data area.

 

Remember: A warning message and the Feedback Status “Local Assigned” indicate that a Feedback copy exists for this facility/project.  When a Feedback copy exists, any changes made to the data area are reflected on the Feedback copy.  If a feedback copy has been returned (Feedback Status “State Review Requested”), this symbol indicates that a Feedback Fact Sheet includes changes to this data area.  Click it to view a pop-up box containing Feedback Fact Sheet.  

 

The DEP lists all available pollution problems.  

 

• Refine the list of available pollution problems by selecting a Pollution Problem Category from the drop down menu (Figure 12-1).  This limits the options in the Available Problems by Category list to just problems in the selected category.  See Table 12-1 for a list of the pollution problems included in each category.

 

 

Figure 12- 1: Pollution Problems Category Dropdown Selections

 

 

 

• To assign a pollution problem category to the facility/project, select the desired pollution problem(s) from the Available Problems by Category list and click to move it to the Selected Problems by Category list.  To remove a problem from the list, select the problem from the Selected Problems by Category list and click to return it to the Available Problems by Category list (Figure 12-2).  Use the control key to select multiple Problems.

 

Click to save all changes to this data area.  This will perform all error checking.  Error messages will appear if entered data does not meet all the business rules.  Click to return to the most recently saved version of the data.

 

Warning: If the user does not click before leaving this data area, entered data is not retained.
