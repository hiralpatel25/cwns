Highlights of the Data Entry Portal (DEP)

The CWNS 2008 DEP was created by migrating the CWNS 2004 “client-server” data entry system into EPA’s web-based Portal environment, adding many significant upgrades (Figure 4-1).  The upgrades include:

 

• Expanding system access from State CWNS Coordinators to local and secondary State users;

 

• Incorporating document upload, storage, and annotation functions;

 

• Generating pre-populated small community forms and receiving these forms marked up by small communities and attached to records in the DEP;

 

• Eliminating some data entry requirements; and

 

• Integrating the DEP with several data entry systems and tools to reduce the effort for submitting and updating data.
