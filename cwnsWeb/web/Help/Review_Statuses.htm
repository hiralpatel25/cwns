Review Statuses

Review Statuses are assigned to each facility/project in the CWNS DEP to indicate the stage that the facility/project has reached in the CWNS process.  Some of these Review Statuses are assigned by State or Federal Users, while others are assigned automatically by the CWNS DEP.  The Review Status also determines whether facility/project data can be edited and who may edit it.  The CWNS 2008 Review Statuses are listed in Table 2-3, and a flow chart depicting the relationship among Review Statuses and Feedback Statuses is provided as Figure 2-1.

 

 

Table 2-3: CWNS 2008 Review Statuses

 

CWNS 2008 Feedback Status

Explanation of Feedback Status

State Assigned

(Headquarters Interim/HI)

Facilities/projects migrated from the 2004 CWNS database begin with this Review Status. The facility/project is available to the State for editing.

State In Progress

(State Interim/SI)

The State User begins editing the facility/project.  The Review Status will automatically change from State Assigned to State In Progress the first time the State User updates the facility/project.

State Deleted

(Deleted/DE)

The State User assigns this Review Status to prevent the facility/project’s data from being changed and submitted to EPA for Review.

Federal Review Requested (State Happy/SH)

The State User with appropriate Access Levels completes all the desired edits and is ready to submit the data for EPA review. The State User cannot assign this Review Status until there are no Data Entry Errors (i.e., there are no checkmarks in the Err column of the Summary portlet).  Once this Review Status is assigned the State User cannot change any data associated with the facility/project until it is reviewed and assigned either State Correction Requested or Federal Accepted.  Annotated documents associated with the facility/project must be submitted either electronically or by mail (see Chapter 16).

State Correction Requested

(Headquarter Rejected/HR)

EPA, or their designated contractor, reviews the data and finds that it does not meet all the CWNS eligibility and documentation criteria.  The reviewer provides information about needed changes (see Review Comments portlet). The State User can now edit the data.

Federal Review Correction

(State Corrected/SC)

The State User completes all the requested edits and is ready to submit the data for subsequent EPA review.

Federal Accepted

(Headquarters Accepted/HA)

EPA, or their designated contractor, reviews the data and finds it acceptable.
