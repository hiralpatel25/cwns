Ask WATERS

Prior CWNS data is available to the public via Ask WATERS (http://www.epa.gov/waters/tools/ask_waters/index.html).  Ask WATERS provides easy menu driven querying, as well as advanced custom querying, of CWNS data.  Results are returned in summary graphs and charts and detailed tables.  Once logged into the DEP, users are able access Ask WATERS through a link on the Main Menu (Chapter 5).
