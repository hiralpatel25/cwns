Administration Portlet

The Administration portlet on the CWNS Welcome Page provides the user access to administrative functions, including changing the user’s own password for the EPA Portal/CWNS DEP and user management (Figure 6-1).

 

• Click Change Password to be redirected to the EPA Portal’s login and password reset page.  

 

Note: For a user with an EPA email account, the EPA Portal user name and password are the same as the network user name and password. If you forget the user name or password, please contact the appropriate EPA network administrator. Do not use this link to reset the user name and/or password.

 

The User Management data area is accessed via the Administration portlet.  This area is available only to State User Managers.

 

• Click the link Manage Users (# Pending) in the Administration portlet to open the User Management Data Area (see next section).

 

The number Pending indicates how many users have applied for CWNS user privileges and are waiting to have their request approved or denied.

 

This portlet also indicates the current Role of users with more than one role.  The current role is indicated by green highlighting (e.g., CA State User in Figure 6-1) and other available roles are hyperlinked (e.g., HQ Federal User in Figure 6-1).  Most users have only one role.  State coordinators may want to create Local roles for themselves in order to better understand the various State roles and practice using State screens for training purposes.

 

• Click the hyperlinked Role to change roles.
