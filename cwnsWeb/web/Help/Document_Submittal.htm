Document Submittal

Documents can be submitted either electronically via the DEP or in hard copy.  Both electronic and hard copy documents must be annotated (see next section).

 

Remember: Footnotable documents can be retained in State’s file. A document is footnotable if a facility’s total needs are less than $20 million (2008 dollars) and is pre-approved as footnotable (Chapter 2 and Appendix C).  Copies of a portion of each State’s footnotable documents will be requested during the Footnotable Documents Audit (Chapter 17).

 

Scanned copies of many of the documents that were submitted for CWNS 2004 and CWNS 2000 are available in the CWNS Portal Library (Chapter 5).  States can only view documents from their own State.  The CWNS Portal Library can be accessed by clicking the Library link on Main Menu of the DEP. For instructions on how access the Library from outside the DEP, see Chapter 3.  Scanned files are in the main folder “State Specific Files.”  Each State has its own subfolder.  Most files are named by eleven digit CWNS number.  The CWNS number is sometimes followed by the letters A, B, C, etc. when multiple documents exist for the same facility/project.  Files that could not be associated with a CWNS number were given a name that begins with the two character State FIPS code followed by a very brief description of the document (e.g., 17 City of Edina; 32 IUP.) or the word “unnumbered.”  Note:  Some documents may contain dark pages due to problems during scanning.

 

To submit documents electronically:

 

• Click next to the Document Title to load or reload (if an electronic version already exists) an electronic document into the DEP.  The reload function is not available if document is linked to another facility/project with a status of Federal Review Requested, Federal Review Correction or Federal Accepted.

 

The Electronic document can be in any Microsoft Office format, WordPerfect, or PDF. The file is converted to PDF by the DEP.

 

• Click to launch a browser providing access to all files available on the computer.  Select the desired file and click (Figure 9-4).  The browser closes and the file name is now listed in the field “Document Files.”  Click to upload the file into the DEP.  A progress bar indicates that the document is uploading.

 

The icon next to the Document Title indicates that the electronic file has been successfully uploaded.  Click to launch the EDMS  to view and annotate the document (Figure 9-5).

 

Hard copy(ies) of document(s) must be submitted to the EPA contractor within one week of assigning the facility/project the Review Status “Federal Review Requested” in the DEP.  Documents must be annotated (See Document Annotation).
