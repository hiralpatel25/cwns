Generating Feedback Copy

In order for a Local User to provide feedback, he/she must have the Facility Update-Partial and/or Facility Update-Feedback access level(s) and a State User must generate a feedback copy for the facility/project (Chapter 5).  

 

• Select and open the desired facility/project from the Facility/Project List

 

• In the Review Status portlet, set the Feedback status to Local Assigned.  A warning message appears on each portlet indicating that “A Feedback version already exists for this facility. Any changes will not be reflected on the Feedback version.”

 

Alternatively, multiple facilities/projects can be assigned Local Assigned status using the link Local Assigned in the Facility/Project List.
