EPA Portal Library

CWNS Portal Library (Figure 3-4) contains the following folders and materials:

 

• “CWNS 2008 Data Entry” contains documents relevant to data entry, including the User Administration Policy and CWSRF Process Certification.

 

• “CWNS 2008 Workgroup” contains Workgroup and Subcommittee minutes, agendas, and other meeting materials.

 

• “Needs Documentation Support” includes information on pre-approved innovative methods, sample documents for most of the pre-approved document types, CWSRF resources, model surveys, and best practices documents.

 

• “Web Seminar Presentations” has the PowerPoint files for all the web seminar trainings conducted in preparation for CWNS 2008.

 

• “State Specific Files” contains documents submitted for CWNS 2000 and 2004, including EPA scanned versions of State-submitted paper documents, and 2004-2008 data migration files.  A State can only access its own State’s files.  Electronic documents from 2000 and 2004 can be uploaded into the DEP to document needs and costs, when appropriate.

 

• “ZZZ- Archives, CWNS 2004” contains the information from CWNS 2004 that was previously accessible via Tetra Tech’s CWNS website.

 

Direct links to the EPA Collaboration Site and Library are available on the Main Menu portlet in the DEP (Chapter 5).
