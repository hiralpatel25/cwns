Footnotable Document Audit

Once all of a State’s facilities needs have been finalized, the EPA contractor randomly audits footnoted documents.  Documents that are selected for the audit must be provided to the EPA contractor.

 

A document is footnotable if a facility’s total needs are less than $20 million (2008 dollars) and is one of the following document types (descriptions of document types are in Appendix C):

 

• Intended Use Plan (1)

 

• State and Federal Loan and Grant Applications (2)

 

• CWSRF Loan Applications (3)

 

• State Needs Surveys & other State forms (12)

 

• Capital Improvement Plan (CIP) (20)

 

• Facility Plan (21)

 

• Final Engineer's Estimate (23)

 

• Sewer System Evaluation Documents (24)

 

• Approved CSO Long-Term Control Plan (LTCP) (33)

 

• Section 319 Funded or EPA Reviewed Watershed-Based Plans (41)

 

• Approved State 319 Project Implementation Plans (43)

 

• Small Community Needs Form (71)

 

The EPA contractor separates the documents into three groups for auditing (Table 17-1).  For each State, the EPA contractor prepares a report documenting:

 

• The dates the audited documents were requested and received   

 

• The number and percent of documents that passed the audit in each category

 

• A narrative description of deficiencies found during the audit

To pass the audit, 75% of the facilities/projects within each group must meet the review criteria.  When a State does not meet the 75% pass threshold in a document type group(s), then one of the following occurs:

 

• If the State finalized needs for 75% of their facilities/projects (based on the number of facilities submitted in 2004) by November 14, 2008, it is given a chance to make corrections and have a second audit of document type group(s) that did not pass.  If the State does not pass the second audit, all needs in the document type group(s) are moved to Unofficial Needs.

 

• If the State did not finalize needs for 75% of their facilities/projects by November 14, 2008, all needs in the document type group(s) are moved to Unofficial Needs.
