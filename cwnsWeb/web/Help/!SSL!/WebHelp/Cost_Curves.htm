Cost Curves

The data area Cost Curves allows users to associate Capital Costs with documents approved for documenting needs and already entered in the Needs by Document area. The Cost Curves are used when documented costs are not available. Cost Curves are available in the DEP for the following Facility Types:

 

• Treatment Plant

 

• Clustered Systems (Decentralized Wastewater Treatment Systems)

 

• Onsite Wastewater Treatment Systems (Decentralized Wastewater Treatment Systems)

 

• Collection: Combined Sewers

 

• Collection: Separate Sewers

 

If no document has been selected, the message “Please select a document” is displayed in the Cost Curve data area.  

 

• Select the document which will be associated with the cost curves from the Needs by Document data area by clicking on the hyperlinked document name.  For CSO Cost Curves, select the State’s CSO Cost Curve document (98) (see Document Search in this Chapter for information on assigning the document type 98).  This allows the user to view the available Cost Curves for the selected document (Figure 9-7).  

 

Only the Cost Curves that are applicable to the facility/project, based on its Facility Types and Changes, are displayed in the Cost Curve data area (Tables 9-5, 9-6, and 9-7).  The Cost Curves are disabled if the Document Type is not approved for Needs.  They are also disabled when Documented Costs have already been assigned to the facility/project in the same Category.  If no Cost Curves are available, the message “No Cost Curves are Available for selection” is displayed in the Cost Curve data area.  

 

• To assign a Cost Curve to the selected document, check the box in the Assigned column.  Click to associate the Cost Curve with the document.

 

A check in the Cost Allocated column indicates that the Cost Curve has been run successfully.  The allocated costs are displayed in the Needs by Document and Capital Costs (indicated by Cost Type C) data areas.

 

If the Cost Curve cannot be run due to data errors, the column Error displays the hyperlinked name(s) of the data area(s) that are the source of the error(s).  Click on the data area(s) name(s) to open the data area page (Figure 9-7).  In the data area, Cost Curve errors are indicated by the prefix CCV.  The Summary portlet displays a in the CC Err column to indicate the source(s) of the error.

 

As soon as all CCV errors are corrected, the Cost Curve is run.   does not need to be clicked again in the Cost Curve data area.  A check in the Cost Allocated column indicates that the Cost Curve has been run successfully.  The allocated costs are displayed in the Needs by Document and Capital Costs (indicated by Cost Type C) data areas.

 

• To delete an assigned Cost Curve, uncheck the box in the Assigned column.  Click to delete any population and cost associated with the Cost Curve and disassociate the Cost Curve from document.  The user receives a confirmation message: Cost will be deleted where Cost Curves are unassigned. Continue. Select OK to unassign the Cost Curve or Cancel to stop the action.
