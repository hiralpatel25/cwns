CWNS 2008 Welcome Page

After logging in, the user is directed to the CWNS 2008 Welcome Page, which is the gateway to the DEP.  From the Welcome Page, users can:

 

• View their existing facilities and projects in the Facility/Project List

 

• Search for and open existing facilities and projects

 

• Create new facilities and projects

 

• Access CWNS resources including the Library, Web Conferencing Service, EPA Collaboration Site, and Ask WATERS Reports

 

• Access administrative functions

 

• Obtain summary Review Status and Feedback Status statistics

 

• Read announcements from EPA
