Permits

The data area Permits allows users to view, edit, and associate new permits with the facility/project.  Existing Permits are listed in ascending order by Permit Number.  

 

Generally, permits are either National Pollution Discharge Elimination System (NPDES) permits or “other” types of permits (e.g., land application, State).  NPDES permits are indicated with a in the column NPDES.  NPDES permits can only be added using the NPDES Permit Search portlet (see next section).

 

Remember: A warning message and the Feedback Status “Local Assigned” indicate that a Feedback copy exists for this facility/project.  When a Feedback copy exists, any changes made to the data area are reflected on the Feedback copy.  If a feedback copy has been returned (Feedback Status “State Review Requested”), this symbol indicates that a Feedback Fact Sheet includes changes to this data area.  Click it to view a pop-up box containing Feedback Fact Sheet.

 

For “other” permits:

 

• Click Add Permit to add a new Permit to the facility/project.  Click in the Edit column to edit an existing Permit.  Either action opens the Add/Edit Permit data area (Figure 7-5).  Explanations of the fields in this data area are in Table 7-3.

 

• Click in the Delete column to remove a Permit from the list.

For NPDES permits:

 

• Click the hyperlinked Permit Number in the data table to launch the NPDES Permit Information in a pop-up box (see next section for details).

 

• To add a NPDES permit use the NPDES Permit Search (see next section).

 

• Clickin the Edit column to open the Add/Edit Permit area.  All data fields for NPDES permits are view-only except the check box “Use Data.”

 

• Check “Use Data” to use data from Envirofacts/PCS to populate the Coordinates, Facility Address, and Primary County fields.  Previously entered data is overridden. Also, the Point of Contact field is updated if no data has been entered in the following fields: authority name, city, and/or State.  If data is entered in these fields, it is not overwritten.  If “Use Data” is indicated for a second Permit, the DEP automatically drops the first “Use Data.”

 

Primary County fields.  Previously entered data is overridden.  Also, the Point of Contact field is updated if no data has been entered in the following fields: authority name, city, and/or State.  If data is entered in these fields, it is not overwritten.

 

• Click in the Delete column to remove a Permit from the list.  If the “Use Data” is selected on the deleted facility, the Coordinates’ source and Facility Address’s source is changed to manual.
