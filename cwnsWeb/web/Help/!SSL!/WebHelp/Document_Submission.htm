Document Submission

An individual facility/project can be submitted as soon as it is completed.  The final day to submit a facility/project for review is October 27, 2008.  

 

Complete the following steps before submitting the facility/project for Federal Review.

 

1. Ensure that that data contains no errors.  The Summary portlet indicates errors by a  in the column Err.  Error messages in each data area describe the error(s). A facility/project cannot be submitted until it is error free.  The Review Status portlet does not have the option submit for Federal Review when there are data entry errors.

 

2. Review any local feedback data.  This symbol in one or more data area, the Feedback Status State Review Requested, and a Feedback Fact Sheet in the Facility/Project Menu portlet all indicate that a Feedback copy exists for the facility/project.

 

• In any data area: Click  to view a pop-up box containing the data in the feedback area.

 

• In the Facility/Project Menu:  Click to launch the HTML version and to launch the PDF version of the Feedback Fact Sheet.

 

Once the facility/project is submitted for Federal Review, the user cannot change any data associated with the facility/project until it is reviewed and assigned either State Correction Requested or Federal Accepted.

 

3. Provide proper supporting documentation.  Documents can be submitted either electronically via the DEP or in hard copy.  Only documents that meet the footnotable document rules can be retained in the State’s file.  All documents (electronic and hard copy) submitted to the EPA contractor for review must be annotated.  Submitted documents (and their corresponding needs) not complying with the annotation guidelines will not be reviewed (Chapter 9).

 

4. Change the Review Status to Federal Review Requested.  In the Review Status portlet, select Federal Review Requested.  Multiple facilities/projects can be assigned Federal Review Requested status using the link Federal Review Requested in the Facility/Project List (Chapter 5).  This Review Status is only available if the facility/project is error free and the State User has Submit for Federal Review access level.
