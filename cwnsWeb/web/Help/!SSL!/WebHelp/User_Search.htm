User Search

The User Search portlet allows the State User Manager to search the User List on one or more of the criteria provided in Table 6-2.

 

 

Table 6- 2: Explanations of User Search Criteria

 

Search Criteria

Explanation

Name

Performs a keyword search on the data field Name

 

Type the keyword(s) in the text box.

Location

Limits the search results to the user’s state.

Type

Limits the search results by User Type.

 

Select All, Local, or State from the dropdown list.

Status

Limits the search results by User Status.

 

Select All, Active, Inactive, or Pending from the dropdown list.

Access Levels

Limits the search results by user Access Level.

 

Select one or more Access Levels from the list.  Use the Control key to make multiple selections.

 

Search results are displayed in the User List.  To clear the search results, click Display Default List.

 

New user accounts can be either initiated by:

 

• The State User Manager, for example he/she creates accounts for State CWNS staff, contractors, and past CWNS participants.

 

• Potential CWNS users of the CWNS community, for example a wastewater treatment facility or municipal government requests access in order to submit data electronically to the State.
