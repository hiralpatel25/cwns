Collection: Combined Sewers Cost Curves

The Collection: Combined Sewers Cost Curve is only enabled if the facility has the facility type Collection: Combined Sewers and meets the following criteria:

 

• Status indicated on the CSO tab is either Requires a Cost Curve or Both Documented and Cost Curve Needs (Chapter 13)

 

• Document type 98 (CSO Cost Curve) is selected in the Needs by Document data area

 

• No out-of-State facilities are included in the sewershed

 

The available CSO Cost Curves are dependent on the Change Type (Table 9-7).

 

When a CSO Cost Curve is assigned and has an allocated cost, the hyperlink details is displayed.

 

• Click details to launch pop-up window with Category V costs for all facilities in the current facility’s Sewershed with a Combined Sewer with status “Requires a Cost Curve” or “Both Documented and Cost Curve Needs.”

 

Note:  The Small Community Form contains a section “Optional Cost Calculation for Sewer Replacement / Rehabilitation Costs.”  The information from this section can be used to estimate population for the CSO cost curves using the following equation:

 

Population = (0.042) * (Feet of Pipe) * (% Replacement or % Rehabilitation)
