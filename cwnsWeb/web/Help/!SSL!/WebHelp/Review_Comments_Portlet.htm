Review Comments Portlet

The Review Comments Portlet is where communication occurs between the State User and the Federal Reviewer.  State Users also use this portlet to communicate with the Secondary State and Local Users when a feedback copy exists (Figure 5-12).

 

• Select the radio button Survey to send comments to and/or view comments from the Federal Review Team.  

 

• Select the radio button Feedback to send comments to and/or view comments from Local and Secondary State Users.  The radio buttons are disabled if there is no feedback copy for the facility/project.

 

The portlet displays the first four Review Comments for the facility/project. The total number of Review Comments is indicated at the bottom of the portlet (e.g. 1-1 of 1).  Each comment includes the first and last name of author, date, time, and the first two lines of the comment.

 

• Click More…> to expand the window and view the entire comment.  Click <Collapse to minimize the window.

 

• To navigate the Review Comments, click Next to display the next four Review Comments and click Prev to display the previous four Review Comments (if not at the beginning of the list).  Click >> to display the last page of Review Comments and << to display the first page of Review Comments. Click View All to display all the Review Comments.  

 

• To add a new Review Comment, click Add Comment and an area to type your comments opens.  After typing your comment, click .  Click Hide to close the comment area without adding a comment.

 

State Users with Facility Update or Submit for Review Access Levels can add Survey comments during the data entry period if the Facility Review Status is State Assigned, State In Progress, or State Correction Requested.  

 

State Users with Facility Update or Submit for Review Access Levels can add Feedback comments during the data entry period if the Feedback Status is Local Assigned or Local In Progress.
