Review Status Statistics Portlet

The Review Status Statistics lists all the Review Statuses (see Chapter 2, Table 2-3 for Review Status definitions) and Feedback Statuses (see Chapter 2, Table 2-4) and indicates the number of the user’s assigned facilities/projects with each status (Figure 5-5).  Review Status focuses on the interaction between the State CWNS Coordinator and the EPA CWNS Review Team, and Feedback Status focuses on the interactions between the State CWNS Coordinator and local or secondary State Users.  The Total row indicates the total number of facilities/projects with Review Statuses assigned to the user, excluding Deleted facilities/projects.  

 

• Click on the hyperlinked number in the Total column to filter the Facility/Project List to display only facilities/projects with the selected Review Status or Feedback Status.
