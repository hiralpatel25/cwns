Summary Portlet

For the selected facility/project, the Summary Portlet provides a synopsis of the facility/project data including:

 

• A list of data areas applicable to the facility/project based on Facility/Project Type

 

• For each data area, an indication whether data is required, is entered without errors, contains errors, and, if applicable, requires correction and contains cost curve errors.

 

• The names of the State data entry personnel and Reviewer

 

• A link to all the technical comments

 

• A link to the facility’s sewershed, when applicable

 

An explanation of the data fields in the Summary Portlet is included as Table 5-4.

 

 

Table 5- 4: Explanations of Summary Portlet Data Fields

 

Data Field

Explanation

Data Area Name

Lists the data areas that are relevant for the facility/project.  For example, wastewater treatment facilities have more data areas and different required data areas than the NPS projects.  Figures 5-8 and 5-9 show how these differences are displayed in the Summary portlets.

Required (Req)

A check mark under this column heading indicates that the data is required for the facility/project.

Entered (Ent)

A check mark under this column heading indicates the data is complete and contains no errors.

Errors (Err)

A check mark under this column heading indicates the data entered into the data area contains errors.  A facility/project cannot be submitted for Federal Review until there are no errors (i.e., no checks in the column Err).

Correction (Cor)

A check mark under this column heading indicates that the Reviewer is requesting a correction in this data area.  This column is only shown when the facility/project has one of following Review Statuses: Federal Review Requested, Federal Review Correction, State Correction Requested, or Federal Accepted.

CC Error

A check mark under this column heading indicates the data entered into the data area is causing a Cost Curve error.  The Cost Curve cannot be successfully run there are no errors (i.e., no checks in the column CC Error).  This column is only shown when the facility/project has one or more Cost Curves (Figure 5-9).

view all comments

Click to this hyperlink to launch a page with the Technical Comments from all the data areas (Figure 5-9).  For each comment, the technical area, and the date of the comment is shown.  Click Close to exit the comments.

View Sewershed

A sewershed is the network of sewers, collection systems, and wastewater treatment systems that are connected to serve a particular area.  For example, when Facility A discharges to another facility (Facility B), then Facility A and Facility B are considered a sewershed.

 

Click this hyperlink to launch a pop-up box with al the names and CWNS numbers of both the present and projected facilities within the same sewershed (Figure 5-10).  Click Close to exit the pop-up box

State

Identifies the first and last name of the most recent State user to update the facility/project and the date of the last update.

Reviewer

Displays first and last name of Federal Reviewer who most recently reviewed the facility/project (i.e. changed the Review Status to Federal Accepted or State Correction Requested) and the date of the action.  If the Review Status is State In Progress, the Reviewer field is blank.
