Combined Sewer Overflow

The Combined Sewer Overflow (CSO) tab is enabled if the facility has a facility type of Collection: Combined Sewers (Figure 13-1).

 

Remember: A warning message and the Feedback Status “Local Assigned” indicate that a Feedback copy exists for this facility/project.  When a Feedback copy exists, any changes made to the data area are reflected on the Feedback copy.  If a feedback copy has been returned (Feedback Status “State Review Requested”), this symbol indicates that a Feedback Fact Sheet includes changes to this data area.  Click it to view a pop-up box containing Feedback Fact Sheet.  

 

 

Figure 13- 1: Combined Sewer Overflow (CSO) Data Area

 

 

 

• Select one of the following CSO statuses from the dropdown list: No needs, problem solved; Documented Needs only; Requires a Cost Curve; Both Documented and Cost Curve Needs

 

The selected CSO Status determines which additional fields are required, optional, and disabled fields (Table 13-1).  Required fields are indicated by a red asterisk (*) and disabled fields are grayed out.

 

• Enter a value greater than or equal to zero in all required fields and desired optional fields.
