CWNS 2008 Categories

Needs are collected in twelve categories (Note: There is no category XI).  For some Needs Categories, official needs only include publicly-owned facilities/projects, while others include both publicly- and privately-owned facilities (Table 2-1).  Detailed descriptions of the Needs Categories are included as Appendix B.
