Announcements Portlet

The Announcement area displays the ten most recent announcements from EPA to the user community.  Examples of the types of announcements included in this data area are changes to the DEP (bug fixes, enhancements), clarifications on the user manual, notice that a Local User has requested access to facility/project, and indication that a new Small Community Form fax is available (Figure 5-6).
