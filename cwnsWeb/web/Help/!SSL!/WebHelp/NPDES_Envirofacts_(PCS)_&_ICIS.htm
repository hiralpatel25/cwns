NPDES Envirofacts (PCS) & ICIS

For the roughly 50% of CWNS records that have associations with one or more NPDES permits, select permit data will be incorporated into the CWNS 2008 DEP from Envirofacts (for PCS, data) and from ICIS (Chapter 8).  State NPDES data is undergoing a multi-year, staged migration from PCS to ICIS.

 

The NPDES permit data in the DEP is updated monthly from Envirofacts/ICIS.  NPDES data will be incorporated into the CWNS DEP via two methods:

 

1) Copied directly into the data entry screens (e.g. Location Data, Point of Contact data) (Chapter 7)

 

2) Summarized in data entry portlets (e.g. NPDES Permit Flow Data) (Chapter 10)

 

The “CWNS 2008 Data Entry” folder in the CWNS Portal Library (Chapter 3) contains a report on data completeness for many of the NPDES data elements utilized in CWNS.
