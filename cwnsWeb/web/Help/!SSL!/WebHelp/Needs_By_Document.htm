Needs By Document

The data area Needs by Document allows users to view, edit, and associate documents with the facility/project.  It allows users to submit electronic copies of the documents.  

 

Remember: A warning message and the Feedback Status “Local Assigned” indicate that a Feedback copy exists for this facility/project.  When a Feedback copy exists, any changes made to the data area are reflected on the Feedback copy.  If a feedback copy has been returned (Feedback Status “State Review Requested”), this symbol indicates that a Feedback Fact Sheet includes changes to this data area.  Click it to view a pop-up box containing Feedback Fact Sheet.

 

Documents associated with the facility/project are listed alphabetically in the data table by Document Title.  The table includes information from data entered into the Add/Edit Document, Capital Costs, and Cost Curves data areas.

 

Note:  When the facility/project data is view-only, a column “Details” is shown in the data table.  Click the to view all the data fields listed in Table 9-1.

• Click Add Document to associate a new document with the facility/project.  Clickin the Edit column to edit an existing Document. (Note: The document type Impaired Waters or TMDL Listing is view only).  Both actions open the Document Details data area (Figure 9-2).  Explanations of the fields in this data area are in Table 9-2.

 

Documents can also be added using the Document Search portlet described later in this Chapter.

 

• Click in the Delete column to remove a Document from the list.  If costs exist for the document, the cost information is also deleted.  A warning message is displayed before the deletion occurs.  Click OK to continue with delete or Cancel to abort the action.  When Impaired Waters or TMDL Listing documents are deleted, the associated information in the Pollution data area is also deleted.

Engineer’s Certification of Outdated Document

The Engineer’s Certification is a letter or a signed and dated memorandum from a:

 

    • Registered Professional Engineer (qualified State staff engineer or independent consulting engineer) to update the outdated documentation of a facility's need and/or cost.

 

    • Local government official to update the need information only. If applicable, Cost Curves can be used to estimate costs.

 

The letter/memo must provide specific information about any changes in need due to population changes, new regulations, projects already completed, or other changes that impact need and/or cost.

Base Month/Year of Cost Information

The Base Month/Year indicates the date of the costs in the document. The date should not be the date on which the source document was published, but the month and year base for the costs. it is not necessarily the same as the publish date. Do not enter inflated costs; the database automatically adjusts the costs using the dates entered into the system.

 

Either enter the date in the mm/yyyy format or click to launch a calendar and select a date.  The DEP calculates the Adjusted Amount in January 1, 2008 dollars.

 

If the Base Month/Year is changed, the Adjusted Amount is recalculated for all Capital Costs, regardless of the facility/project, associated with the document.  The Base Month/Year cannot be changed if the document is assigned to another facility/project with the review status of Federal Review Requested, Federal Review Correction, or Federal Accepted.
