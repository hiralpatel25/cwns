Facility / Project Advanced Search Portlet

The Advanced Search portlet in the left column allows users to search their existing Facilities/Projects on one or more of the criteria provided in Table 5-1.  This portlet is minimized by default.

 

• Click to maximize the Advanced Search Portlet and to minimize it again.

 

 

Table 5- 1: Explanations of Advanced Search Criteria

 

Search Criteria

Explanation

Review Status

Limits the Facility/Project List to the selected Review Status(es).

 

Select one or more Review Status from the list provided.

 

Use the Control key to highlight more than one Review Status.

Facility/Project Name

Performs a keyword search on the data field Facility/Project Name.

 

Type the keyword(s) in the text box.

Overall Type

Limits the Facility/Project List to the selected Overall Type(s).

 

Select one or more Overall Type from the following: Wastewater, Decentralized Wastewater Treatment, Point Source, Nonpoint Source, Stormwater, Plan Development.  Use the Control key to highlight more than one Overall Type.

Authority

Performs a keyword search on the data field Authority.

 

Type the keyword(s) in the text box.

System Name

Limits the Facility/Project List to the selected System Name.

 

Type keyword(s) in the text box to perform a keyword search on the System Name this data field.

 

Click  to launch the System Name Lookup and view the list of System Names currently in the DEP.  

Click the radio button of the desired System Name and then .  The selected System Name is populated in the text box.

County/FIPS Code

Limits the Facility/Project List to the selected County.

 

Type keyword(s) in the text box to perform a keyword search on the County name.

 

Click  to launch the County/FIPS Code Lookup and view the list of all Counties/FIPS Codes in the State.  

Click the radio button of the desired County/FIPS Code and then .  The selected County/FIPS Code is populated in the text box.

Watershed

Limits the Facility/Project List to the selected Watershed.  

 

Type keyword(s) in the text box to perform a keyword search on the Watershed name.

 

Click to launch the Watershed Lookup and select view the list of all Watersheds in the State.  Click the radio button of the desired Watershed and then .  The selected Watershed is populated in the text box.

Document Type

Limits the Facility/Project List to the selected Document Type.  

 

Select one or more Document Type(s) from the list provided.  

 

Use the Control key to highlight more than one Document Type.

Total Needs

Limits the Facility Project List to facilities/projects that are greater than, less than, or equal to the entered Federal Adjusted Need Amount.  

 

Select either > (greater than), < (less than), or = (equal to) from the dropdown list under the heading Total Need.  Then enter a numeric amount in the text box as an integer (no commas, no decimal points).

 

Remember: The search is based on Federal needs adjusted to January1, 2008 dollars.

Documentation

Limits the Facility/Project List to facilities/projects with documentation entered before, after, or on the specified date.  This search is useful for identifying facilities/projects that need to be redocumented for CWNS 2008 (See Documentation Rules in Chapter 2).

 

Select either > (after), < (less than), or = (before) from the dropdown list under the heading Documentation.  Then enter the date in the mm/dd/yyyy fomat or click to launch a calendar and select a date.

Present Population

Limits the Facility/Project List to facilities/projects with specified population values.

 

Select from the dropdown list: Residential Receiving Collection, Residential Onsite Wastewater Treatment, Residential Clustered Systems, Non-residential Receiving Collection, Non-residential Onsite Wastewater Treatment, Non-residential Clustered Systems.  

 

Select either > (greater than), < (less than), or = (equal to) from the dropdown list under the heading Present Population.  Then enter the number of persons as an integer (no commas, no decimal points).

Present Flow

Limits the Facility/Project List to facilities/projects with specified present flow values, in MGD.

 

Select from the dropdown list: Municipal, Industrial, Wet Weather Peak, Total, Infiltration.  Select either > (greater than), < (less than), or = (equal to) from the dropdown list under the heading Present Flow.

 

Then enter the flow in MGD as an integer (no commas, no decimal points).

Facility / Project Last Changed

Limits the Facility/Project List to facilities/projects that have not been edited since the entered date.  This search is useful for identifying facilities/projects that have not been updated since CWNS 2004.

 

Select either > (after), < (before), or = (on) from the dropdown list under the heading Facility/Project Last Changed.  Either enter the date in the mm/dd/yyyy fomat or click to launch a calendar and select a date.

Permit Number

Limits the Facility/Project List to facilities/projected associated with the entered permit number.

 

Enter the permit number.

State

Limits the Facility/Project List to the selected State. The default selection is the user’s State.  Users search and view other States’ data, except for Needs information.

 

Select the desired State from the dropdown list.

Show Facilities/Projects

The default selection is All.  The “No Errors” and “With Errors” only applies to facilities/projects with the Review Statuses State in Progress and State Assigned.

 

Click the radio button “No Errors” to limit the Facility/Project List to facilities/projects without errors, and click the radio button “With Errors” to limit the Facility/Project List to facilities/projects containing errors.

 

 

• After entering one or more search criteria, click .  The first twenty facilities/projects that meet all the entered search criteria are now listed in the Facility/Project List (Figure 5-2).  The total number of Facilities/Projects that meet the search criteria is indicated at the bottom of the Facility/Project List (e.g. 1-20 of 65).  The header “Facilities based on advance search” indicates that the Facility/Project list has been filtered based on an advanced search.

 

• This refined list can be navigated and sorted in same ways as the complete Facility/Project List.   Note: If the user does another Advanced Search, the Advanced Search is performed against all facilities/projects in the State, not just the refined list.

 

• Click the hyperlink Display All Facilities to return to the list of all facilities/projects.

 

• Click to delete all entered search criteria from the Advanced Search Portal.
