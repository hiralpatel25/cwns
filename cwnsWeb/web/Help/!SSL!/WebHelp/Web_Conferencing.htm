Web Conferencing

Web Conferencing capability is available to State CWNS Coordinators through the EPA portal.  The capability can be used to train Local Users and share screens with local users, EPA, or contractors during the data entry and review process.

 

To use this function:

 

• Enter the portal at http://portal.epa.gov .  Under the heading General Pages, click Team Resources (Figure 3-2).  The Team Resources page includes access to the Web Conferencing.

 

• Click Enter Web Conferencing to begin to use this function.

 

• Click the schedule tab to arrange a Web Conference (Figure 3-5).

 

A direct link to the CWNS workspace of the Web Conferencing is available on the Main Menu portlet in the DEP (Chapter 5).

 

For more information on using the Web Conferencing function, utilize the help function within the system.  The Help link is located in the upper right hand corner of each screen.
