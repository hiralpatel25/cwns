Facility / Project Search Portlet

The Search portlet in the left column allows users to perform a keyword search of their existing Facilities/Projects.  Users can search for text that is part of the Facility/Project Name, Description, CWNS Number, County, and Authority data fields (Figure 5-2).

 

• Type the keyword(s) in the text box and click .  The first twenty facilities/projects that meet the search criteria are now listed in the Facility/Project List (Figure 5-2).  The total number of Facilities/Projects that meet the search criteria is indicated at the bottom of the Facility/Project List (e.g. 1-20 of 65).  The Facility/Project List indicates the terms of the search in its header with the sentence “Facilities based on search keyword: ENTERED KEYWORD (e.g. los angeles).”

This refined list can be navigated and sorted in the same way as the complete Facility/Project List.

 

• Click the hyperlink Display All Facilities to return to the list of all facilities/projects.
