Portal Environment

The EPA Portal provides abilities for local-State-EPA topic-focused communities (e.g. CWNS data entry community) to conveniently communicate and share data, documents, and tools (one user name and password for many communities/systems/tools) in a secure environment (users are limited to data within their community and within their business responsibility).  As of November 2007, there were ten communities in the Portal.

 

The portal also provides very good capabilities to customize the look of the data entry screens based upon type of user and type of data being entered.  This is accomplished through use of “portlets,” small modular windows that:

 

• Can interact with other portlets or function on their own

 

• Be assembled to create a seamless, holistic view of the DEP

 

Each data area (e.g. Needs by Document, Flow Information) in the DEP is an individual portlet.
