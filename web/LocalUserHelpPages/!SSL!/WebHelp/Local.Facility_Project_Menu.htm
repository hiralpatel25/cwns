Facility/Project Menu

The Facility/Project Menu (Figure 5-1) provides links to the:

 

• Facility/Project Fact Sheet containing a summary of the most recent data entered for the facility/project by the State.  If the State has changed the data since generating the Feedback Copy, the data in the Fact Sheet will be different than the data in the Feedback Copy.

 

• Small Community Needs Form for facility/project’s with a total population of fewer than 10,000 persons (Chapter 3)

 

• Feedback Fact Sheet containing a summary of the data in the Feedback Copy, including all changes made by the Local User.  This Feedback Fact Sheet is used by the State to incorporate Local User feedback into the facility/project data.

 

• Click to launch the HTML version and to launch the pdf version the Facility/Project Fact Sheet, Small Community Needs Form, or Feedback Fact Sheet.
