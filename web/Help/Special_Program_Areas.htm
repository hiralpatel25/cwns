Special Program Areas

The list of Special Program Areas includes the study areas of the twenty-eight National Estuary Programs (NEPs) designated under the section 320 of the Clean Water Act.

 

• Click to delete a Special Program Area from the list of associated Special Program Areas.

 

• Click Add Special Program Area to associate a new Special Program Area with the facility/project.  This opens the Add Boundary data area (Figure 8-6).  To add a Special Program Area, click to view an alphabetical list of Boundaries in the State.  Select the radio button of the desired Special Program Area and click .

 

Click to save all changes to this data area.  This will perform all error checking.  Error messages will appear if entered data does not meet all the business rules.  Click to abandon changes.

 

Warning: If the user does not click before leaving this data area, entered data is not retained.
