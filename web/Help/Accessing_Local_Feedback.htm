Accessing Local Feedback

When a local Feedback copy is available for the State to review, the Feedback Status is changed to State Review Requested.  Note: The Advanced Search portlet (Chapter 5) can be used to narrow the Facility/Project list to only facilities/projects with the State Review Request status.  Within a facility/project, a is displayed in each data area where the Local User has changed the data (Figure 15-1).

 

• Click to view a pop-up box containing the Feedback Fact Sheet

 

 

Figure 15- 1: Example of Feedback Form.  Note the in the right-hand corner indicates a change in this data area.  The Feedback Fact Sheet shows the Local User add added a System Name.

 

 

 

The Feedback Fact Sheet can also be launched from the Facility/Project Menu on the Facility/Project Information tab (Chapter 7) (Figure 15-2).

 

• Click to launch the HTML version and to launch the PDF version of the Feedback Fact Sheet.

 

 

Figure 15- 2: Facility/Project Menu

 

 

 

After the State User reviews the Feedback Fact Sheet and integrates changes as appropriate, he/she changes the Review Status to State Applied.  This indicates to both the State and Local User that the feedback has been reviewed.  The State User can no longer access the Feedback Fact Sheet and the symbols are removed from all the facility/project’s data areas.
