User List

The user list provides an alphabetical list by last name of the names and portal usernames of users currently associated with the State (Figure 6-2).  It also indicates the user’s status as:

 

• Active: Currently is granted access to one or more facility/project in the DEP.

 

• Inactive: Not currently granted access to one or more facility/project in the DEP.

 

• Pending: Waiting for requested user access to be approved or denied.  Users with this status have applied for a CWNS user account via the self-registration process (see User Initiated New User Accounts) or have been created by the State as a new user.

 

When there are Pending Users (indicated in parenthesis following the Manage Users link in the Administration portlet), the User List displays only Pending Users when the User Management area is opened.  The User Search can be used to find other Users.

 

The User List displays ten users at a time.  The total number of users is indicated at the bottom of the User List (e.g., 1-10 of 21).

 

• Click Next to display the next ten users (if not at the end of the list) and click Prev to display the previous ten users (if not at the beginning of the list).  Click >> to display the last page of users and << to display the first page of users.

 

See User Search for information on searching the User List.

 

The User List portlet allows users State User Managers to launch the User Details portlet.

 

• Click on the hyperlinked name to open User Details portlet for the selected user.  The selected user is highlighted in Green in the User List (see Editing User Details).

 

• Click the hyperlink Add User to add an additional user to the current State.  This action also opens the User Details portlet (see State Initiated New User Accounts).
