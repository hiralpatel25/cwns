Coordinates

The data area Coordinates allows users to view, update, and enter coordinate information for the facility/project (Figure 8-2).  Data can be sourced from a NPDES permit, entered manually, or generated using the tool WATERS Lite Viewer (WLV).  WLV can also be used to view a map indicating the location of the coordinate data from other sources.
