Review Status Portlet

From the Review Status Portlet (Figure 5-11), State Users with appropriate access levels can:

 

• View the current Review Status and Feedback Status, if applicable, of the facility/project

 

• Submit the facility/project for Federal Review

 

• Assign a facility/project to a Local User

 

• Delete a facility/project

 

See Chapter 2 for definitions of the Review Statuses and Feedback Statuses.

 

 

Figure 5- 11: Review Status Portlet

 

 

 

State Users with Facility Update or Submit for Federal Review access level (Chapter 6) have access to the Review Status Portlet when the Facility Review Status is State Assigned, State In Progress, State Correction Requested, or Deleted during the data entry period.  Only State Users with Federal Review access levels are able to select Federal Review Requested, or Federal Review Correction.

 

In the Review Status portlet, the current Status is displayed (Figure 5-11).  

 

• To update a facility/project’s Review Status, select the appropriate Review Status from the dropdown list and click .

 

The current Status is updated in the system.  Note: The list of available Review Status changes according to the user’s security level and the Current Review Status of the facility/project.

 

• Restore a facility/project with the Review Status Deleted by selecting “Restore” from the dropdown list in the Review Status Portlet and then clicking .   

 

When a facility/project is restored, the DEP checks for errors in all the of data areas, performs Cost Curve validation for all data areas, and reruns all Cost Curves that were run prior to deletion. Any errors detected during this process are indicated in the Summary Portlet.

 

• To submit a facility/project for review, select the Review Status “Federal Review Requested.” Multiple facilities/projects can be assigned Federal Review Requested status using the link Federal Review Requested in the Facility/Project List (see earlier section in this Chapter.)

 

The State User must have Submit for Federal Review Access Level and the following conditions must apply:

 

• There are no Data Entry Errors (i.e., there are no checkmarks in the Err column of the Summary Portlet.).  

 

• For a facility that is part of a Sewershed, all upstream facilities must be Federal Review Requested/Federal Review Correction/Federal Accepted.

 

• For a facility/project documented using the Small Community Form: The State User must confirm that the Small Community Form has been received and has the appropriate signatures.

 

When a facility/project has a feedback copy that has not been applied by the State (review statuses of Local Assigned or Local In Progress), the State User is warned that he/she is not able to apply changes from the Feedback version after facility/project is submitted for Federal Review.  

 

In the Review Status portlet, the “Feedback Status” dropdown is enabled if the Review Status is State Assigned, State In Progress, or State Correction Requested, or if a feedback copy exists for the facility/project. The default “Feedback Status” listed is blank.  

 

State Users with either Facility Update Access Level or Submit for Federal Access Level can create/delete a feedback copy of the facility/project.

 

• To create a feedback copy, select the status to “Local Assigned” in the dropdown list.  Multiple facilities/projects can be assigned Local Assigned status using the link Local Assigned in the Facility/Project List (see earlier section in this Chapter).

 

• To delete a feedback copy, chose the option “Revoke” in the dropdown.
