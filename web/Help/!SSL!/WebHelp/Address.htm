Address

The data area Address allows users to view, edit, and update the address of the facility (Figure 8-1).  Note: Address data is not collected for NPS pollution projects.  Therefore, this data area will not be shown.

 

The Address data area will be in view only mode if the information is sourced from a NPDES permit.  The Source field shows “Envirofacts” to indicate that a NPDES permit is the source data.  This occurs when “Use Data” is selected for a NPDES permit in the Permits data area (Chapter 7).  The user cannot edit the address data in the CWNS DEP.

 

State Users are encouraged to submit updated facility address information to:

 

• Their State’s NPDES data manager, or

 

• For Permit Compliance System (PCS) States only: The EnviroFacts Error Correction Process available at: http://oaspub.epa.gov/enviro/ets_grab_error.smart_form

 

If “Use Data” is not selected for a NPDES permit, the Address area is available to edit and update.  No Source is listed.  Explanations of the fields in this data area are in Table 8-2.

 

Remember: This symbol indicates that a Feedback copy exists for this facility/project.  Click it to view a pop-up box containing the changes made in Location Information Feedback area.
