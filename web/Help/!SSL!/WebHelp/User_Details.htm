User Details

The User Details allows State User Managers to view and change current user information, add new user information, and approve and deny pending users.  If no user has been selected from the User List, the message “Please Select a User From the User List” is displayed in the User Details area.

 

After clicking the hyperlinked user name from the User List, the User Details portlet displays the selected user’s data (Figure 6-8).  The User Details portlet is also displayed at the completion of the State initiated new user process.  Table 6-5 explains the User Details data fields.

 

 

Figure 6- 8: User Details Data Area

 

 

 

Table 6- 6: Explanations of User Details Data Fields

 

Data Field

Explanation

User Details

CWNS User ID

Unique identifier generated for CWNS users by the DEP.  This field is view only.

Name

This user’s name in the format “last name, first name.”  This field is view only.

Status

Select Active from the dropdown list to permit the user to have access to the DEP.  Chose Inactive from the dropdown list to deny the user access to the DEP

Email

The user’s email address.  Enter or edit the user’s email address.

Phone Number

The user’s phone number.  Enter or edit the user’s ten-digit telephone.

Roles

Primary

If there is more than one role, primary indicates which view is shown on the CWNS Welcome Page when the user logins.  Select the radio button of the desired Primary Role.  The change takes place the next time that the user logins.

Role

Indicates the user’s Role(s) (e.g., Federal, State, Local)

Access

Indicates the user’s Access Levels.  See Table 6-1 for an explanation of Access Levels.

 

 

• Click in the Delete column to remove a Role from the user.

 

• Click Add Role to add a new Role to the user.  Clickin the Edit column to edit an existing role.  Both actions open the Role Information data area (Figure 6-9).  

 

To give the user additional Access Level(s), check all the desired Access Level(s).  Uncheck the box to revoke Access Level(s) from the user.  Note:  All CWNS users have View access.  Change the Status to Inactive to deny the user View access.

 

Select the radio button “All Facilities in the State” to allow the user access to all State data.

 

Select the radio button “Selected Facilities” to restrict user access.  Click and a pop-up box with a list of the State’s facilities/projects is launched.

 

Enter a keyword in the box and click to search the list.  The system searches the entered keyword on the Facility/Project Name only.  Delete the keyword and click to clear the search.

 

Check the boxes of the facilities/projects that should be assigned to the user and click .  The facilities/projects are listed in the Role Information area.  

 

Revoke one or more facility/project from the user by highlighting the name and clicking .  Use the control key to select multiple facilities/projects.

 

Remember to click to save all changes made to current user.  Click to abandon changes.
