Data Areas

Each CWNS facility/project is assigned one or more Facility/Project Type (formerly Nature) to describe its basic components.  The Facility/Project Type also dictates which data areas are required (R), optional (O), and not applicable (N/A) (Table 2-5).  The DEP does not display the data areas that are not applicable (N/A).   

 

 

Table 2-5: Required (R), Optional (O), and Not Applicable (N/A) Data by Facility/Project Type

 

* Unit Process data is required for wastewater facilities with a design flow of greater than 10 MGD.

** Coordinates are not required if the facility/project is projected only.  In certain scenarios, other coordinates data options are available for Decentralized Wastewater Treatment Systems, Stormwater Program Management, and NPS Control (See Chapter 8)
