Local Feedback

This symbol is displayed in data areas where a change has been made in the Feedback copy of the facility/project data by a Secondary State or Local User.  

 

• Click to view a pop-up box containing the data in the feedback area.

 

If the facility/project has any of the following statuses, State Assigned, State In Progress or State Correction Requested, and a Feedback copy exists with status of Local Assigned or Local in Progress, a warning message “A Feedback version already exists for this facility.  Any changes will not be reflected on the Feedback version” is displayed in each data area.
