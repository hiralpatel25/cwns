Technical Data Area Pages

When the user opens a facility/project from the Facility/Project List, the Facility/Project data area is displayed.  Tabs across the top of the screen allow the user to access other technical data area pages relevant to the facility/project, and breadcrumbs allow the user to return to the Facility/Project List.

 

Each of the Technical Data Area Pages (Facility/Project, Location, Needs, Population & Flow, Discharge & Effluent, Pollution, Utility Process, and Utility Management) have these portlets on the left side of the page (Figure 5-7):

 

• Summary

 

• Review Status

 

• Review Comments
